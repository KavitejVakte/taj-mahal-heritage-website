# Taj Mahal Heritage Website

A modern animated website showcasing the history, architecture, and cultural legacy of the Taj Mahal.

## Features
- Modern UI design
- Smooth animations
- Interactive gallery
- Historical timeline
- Responsive layout

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was created as part of a college assignment to demonstrate web development skills and present historical heritage using modern web design.
